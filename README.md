# ✨ 이지은 포트폴리오

개인의 기술 스택, 프로젝트 경험, 개발 철학을 효과적으로 전달하기 위해 React 및 TypeScript 기반으로 개발된 반응형 웹 포트폴리오입니다.

## 🚀 1. 핵심 개요 및 배포 (Overview)

| 항목       | 내용                                                                  |
| ---------- | --------------------------------------------------------------------- |
| 프로젝트명 | 이지은 포트폴리오                                                     |
| 개발 기간  | 2025. 11. 01 ~ 2025. 11. 18. ( 포터블 모니터 이슈로 작업시간 지연됨 ) |
| 배포 주소  | https://ijieun0123.github.io/portfolio-v2/                            |

## 🛠️ 2. 기술 스택 (Tech Stack)

이 프로젝트에 사용된 주요 기술 및 라이브러리 목록입니다.

| 항목            | 내용                                            |
| --------------- | ----------------------------------------------- |
| 언어            | TypeScript                                      |
| 프레임워크      | React                                           |
| 스타일링        | Styled-Components, SCSS                         |
| 애니메이션 / UI | AOS, Swiper                                     |
| 버전 관리       | Git, GitHub                                     |
| 패키지 관리     | Yarn                                            |
| 배포            | GitHub Pages (GitHub Actions를 통한 CI/CD 구축) |
